# Security Policy

## Reporting a Vulnerability

We take security seriously at Privasys. If you discover a vulnerability
in this repository, please report it responsibly through one of the
following channels:

- **Email:** [security@privasys.org](mailto:security@privasys.org)
- **GitHub:** Open a [private security advisory](https://github.com/Privasys/wallet/security/advisories/new)
